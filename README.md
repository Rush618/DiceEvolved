# DiceEvolved
A linear, text-based RPG.

Starting the game:

       1) Download & Install Python 3.7+
       2) Download DiceEvolved.zip file
       3) Extract files into convenient folder
       4) Open the "Dice Evolved" file with IDLES
       OR
       4) Run the "Dice Evolved" file
